# chat-client-vue

## Project setup
```
yarn install
```
```
yarn global add @vue/cli concurrently wait-on cross-env
```

### Compiles and hot-reloads for development
```
yarn start
```

### Compiles and minifies for production
```
yarn build
```
or
```
yarn build-win
```
